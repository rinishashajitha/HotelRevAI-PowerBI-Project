# HotelRevAI-PowerBI-Project
AI-Driven Revenue Analysis for Hotels using Power BI (Infosys Internship Project)

## 📖 Project Story

Hotels generate massive amounts of booking data every day, but most of it remains underutilized. This project was built to bridge that gap — transforming raw reservation data into meaningful business insights.

**HotelRevAI** is an end-to-end Power BI solution designed to analyze hotel performance, understand customer behavior, and uncover revenue-driving factors. Instead of just visualizing data, the project focuses on *why* trends happen and *how* decisions can be improved.

---

## 🎯 What This Project Solves

Hotel management often struggles with:

* Unpredictable booking demand
* High cancellation rates
* Ineffective pricing strategies
* Low customer retention

This project addresses these challenges by converting historical booking data into actionable insights using business intelligence techniques.

---

## 🧠 My Approach

### 1. Data Preparation

* Cleaned and transformed raw booking data
* Handled missing and inconsistent values
* Created structured columns like arrival date and revenue

### 2. Data Modeling

* Designed a **Star Schema model** for performance and scalability
* Built a central fact table with multiple dimension tables
* Ensured proper relationships (one-to-many, single direction filtering)

### 3. Business Metrics (DAX)

Developed key performance indicators such as:

* Total Revenue
* Total Bookings
* Average Daily Rate (ADR)
* Average Length of Stay
* Cancellation Rate

These metrics enable dynamic and interactive analysis across multiple dimensions.

### 4. Dashboard Development

Created four focused dashboards:

* **Executive Overview** – High-level KPIs and performance summary
* **Booking Behavior Analysis** – Customer patterns and booking trends
* **Revenue & Pricing Analysis** – Revenue growth and pricing efficiency
* **Operational Insights** – Stay patterns and service-related insights

---

## 📊 Key Insights Discovered

* 📈 Revenue peaked in 2019, followed by a drop in 2020
* 🌐 Online travel agencies are the primary revenue drivers
* 👤 Majority of customers are first-time guests
* 📅 Weekday bookings generate higher revenue than weekends
* ⏳ Short lead-time bookings dominate customer behavior

These insights highlight opportunities for improving pricing strategies, marketing focus, and customer retention.
